关于海伯高斯

海伯高斯是一家专业的招聘管理咨询服务公司，公司致力于为客户提供最有效的人才招聘访寻服务，专注于中国本土以及海外的高端管理人才和海外高新技术人才的搜寻。 海伯高斯专注于高科技和房地产行业的高级人才招聘访寻服务，本着“成就客户，成就员工”的战略发展思想为客户提供专业的招聘咨询服务，为员工提供良好的发展平台。 海伯高斯的猎头顾问平均在猎头行业的工作经验超过8年，公司同时拥有强大的人才数据库以及行业知识库。
海伯高斯会为大家提供一个优秀的成长平台，欢迎志同道合者将个人相关资料发至：hr@hyper-g.com.cn
公司也欢迎喜欢人力资源服务行业的大学生来公司实习。
